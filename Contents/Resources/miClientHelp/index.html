Edit Script

miClient Reference

トップレベルのスクリプトの property として XText が必要。

Document Infomation

document_file_as_alias

ドキュメントファイルのエイリアスを取得します。

document_name

ドキュメント名を取得します。

is_modified

最前面の書類が変更されているかどうか

text_encoding

最前面の書類テキストエンコーディングを取得します。

document_mode

最前面の書類のモード

count_documents

アプリケーションで開かれている書類の数

exists_document

アプリケーションで書類が開かれているか？

Working with Contents

paragraph_at

指定したインデックスの paragraph を取得します。

document_content

書類の内容を取得します。

count_paragraph

すべての行数を数えます。

Working with Selection

selection_contents

選択されているテキストを取得

selection_ref

選択されているテキストをリファレンスを取得

current_paragraph

キャレットがある行のテキストを取得

index_current_paragraph

キャレットがある行番号を取得

set_insertion_point_at

キャレットの位置を変更します。

set_paragraph_at

指定した行のテキストを変更します。

selection_info

選択範囲に関する情報を取得します。

select_in_range

指定した範囲を選択します。

insert_text

キャレットの位置にテキストを挿入します。選択範囲があれば選択範囲を置き換えます。

Save Document

save_document

書類を保存します。

save_with_asking

書類をユーザーに問い合わせてから保存する。
